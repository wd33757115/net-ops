# 公文质量检查清单

## 使用说明

本清单用于公文起草完成后进行质量检查，确保公文格式规范、内容完整、语言准确。请逐项核对，发现问题及时修改。

---

## 一、格式检查清单

### 1. 用纸及版面检查

- [ ] **用纸规格**：是否使用A4型纸（210mm×297mm）
- [ ] **纸张质量**：是否使用60-80g/m²胶版印刷纸或复印纸
- [ ] **装订方式**：是否采用左侧装订，不掉页
- [ ] **天头尺寸**：上白边是否为37mm±1mm
- [ ] **订口尺寸**：左白边是否为28mm±1mm
- [ ] **版心尺寸**：是否为156mm×225mm
- [ ] **印刷方式**：是否双面印刷
- [ ] **页码位置**：版心下边缘，左右各一字线

### 2. 字体字号检查

- [ ] **发文机关标志**：是否使用小标宋体（颜色红色）
- [ ] **文件标题**：2号小标宋体，居中排布
- [ ] **正文**：3号仿宋_GB2312字体
- [ ] **第一层次标题**：3号黑体（一、）
- [ ] **第二层次标题**：3号楷体_GB2312（（一））
- [ ] **第三层次标题**：3号仿宋_GB2312（1.）
- [ ] **第四层次标题**：3号仿宋_GB2312（（1））
- [ ] **发文字号**：3号仿宋_GB2312，六角括号〔〕
- [ ] **附件说明**：3号黑体
- [ ] **附注**：3号仿宋_GB2312，加圆括号
- [ ] **抄送机关**：4号仿宋_GB2312
- [ ] **印发机关和日期**：4号仿宋_GB2312
- [ ] **页码**：4号半角宋体

### 3. 结构层次检查

- [ ] **第一层**：是否使用"一、"，黑体
- [ ] **第二层**：是否使用"（一）"，楷体_GB2312
- [ ] **第三层**：是否使用"1."，仿宋_GB2312
- [ ] **第四层**：是否使用"（1）"，仿宋_GB2312
- [ ] **层次顺序**：是否严格按照"一、（一）1.（1）"顺序
- [ ] **标题回行**：是否为梯形或菱形

### 4. 成文日期检查

- [ ] **格式**：阿拉伯数字全称（如：2025年1月26日）
- [ ] **位置**：发文机关署名下方
- [ ] **月日格式**：不编虚位（如：1月5日，不写01月05日）
- [ ] **右空**：是否右空四字编排

### 5. 发文字号检查

- [ ] **格式**：机关代字+六角括号〔〕+年份+序号+号
- [ ] **年份**：是否4位阿拉伯数字全称
- [ ] **括号**：是否使用六角括号〔〕，不是[]或【】
- [ ] **序号**：是否不加"第"字，不编虚位（如：5号，不写05号）
- [ ] **位置**：发文机关标志下空二行，居中

### 6. 格式要素完整性检查

- [ ] **份号**：如需标注，是否6位3号阿拉伯数字
- [ ] **密级**：如需标注，是否使用3号黑体
- [ ] **紧急程度**：如需标注，是否使用3号黑体
- [ ] **发文机关标志**：是否居中
- [ ] **签发人**：上行文是否标注，格式是否正确
- [ ] **标题**：是否完整、居中
- [ ] **主送机关**：是否顶格，后有冒号
- [ ] **附件说明**：是否在正文下空一行
- [ ] **发文机关署名**：是否正确
- [ ] **附注**：是否加圆括号，在成文日期下一行
- [ ] **附件**：是否另面编排，在版记之前
- [ ] **版记**：首条分隔线在第一个要素之上
- [ ] **页码**：是否连续编排

## 二、内容检查清单

### 1. 标题检查

- [ ] **准确性**：是否准确概括内容主旨
- [ ] **完整性**：发文机关+事由+文种
- [ ] **规范性**：事由是否使用"关于"引出
- [ ] **简洁性**：是否简洁明了，不冗长
- [ ] **适用性**：是否符合文种要求

### 2. 主送机关检查

- [ ] **准确性**：是否为直接上级或相关机关
- [ ] **单一性**：请示是否只有一个主送机关
- [ ] **排序**：多个主送机关是否按规范排序
- [ ] **位置**：是否在标题下空一行顶格
- [ ] **标点**：最后一个机关后是否为冒号

### 3. 正文内容检查

#### (1) 请示类

- [ ] **一文一事**：是否只请求解决一个问题
- [ ] **行文依据**：政策依据、实际情况是否充分
- [ ] **理由说明**：理由是否充分、合理
- [ ] **请求事项**：是否明确、具体
- [ ] **可行性**：请求事项是否可行
- [ ] **附件齐全**：必要的附件是否附上

#### (2) 通知类

- [ ] **通知依据**：制发依据是否充分
- [ ] **通知事项**：是否清晰、具体
- [ ] **执行要求**：是否明确、可操作
- [ ] **时间地点**：会议/活动时间地点是否明确
- [ ] **权责清晰**：责任单位是否明确

#### (3) 函类

- [ ] **发函缘由**：是否说明清楚
- [ ] **商洽事项**：是否明确、合理
- [ ] **答复依据**：是否有政策依据
- [ ] **答复明确**：是否态度明确、不含糊

#### (4) 总结类

- [ ] **成绩全面**：主要成绩是否全面总结
- [ ] **数据准确**：数据是否真实、准确
- [ ] **经验典型**：经验总结是否有亮点
- [ ] **问题准确**：问题找得是否准确
- [ ] **打算可行**：下一步打算是否可行

#### (5) 纪要类

- [ ] **会议信息完整**：时间、地点、主持人、出席、缺席、列席
- [ ] **主要议题清晰**：会议讨论的主要问题是否清晰
- [ ] **各方意见记录**：主要观点是否记录
- [ ] **议定事项明确**：会议决定的事项是否明确
- [ ] **责任分工清晰**：责任单位、时限是否清晰

### 4. 结尾用语检查

- [ ] **请示**：是否使用"妥否，请批示"等规范结语
- [ ] **报告**：是否使用"特此报告"等规范结语
- [ ] **通知**：是否使用"特此通知"等规范结语
- [ ] **函**：是否使用"特此函复"或"请予函复"等规范结语
- [ ] **纪要**：是否简明扼要总结

### 5. 附件检查

- [ ] **必要性**：附件是否必要
- [ ] **完整性**：附件是否齐全
- [ ] **一致性**：附件标题与附件说明是否一致
- [ ] **规范性**：附件格式是否符合要求
- [ ] **顺序号**：多个附件是否有顺序号

## 三、语言检查清单

### 1. 准确性检查

- [ ] **用词准确**：概念是否准确，有无歧义
- [ ] **专业术语**：专业术语使用是否正确
- [ ] **政策依据**：引用政策是否准确，出处是否清楚
- [ ] **数据准确**：数字、数据是否准确无误
- [ ] **逻辑严密**：推理判断是否符合逻辑
- [ ] **语句完整**：句子成分是否完整

#### 常见错误用词核查

- [ ] "截止"是否改为"截至"
- [ ] "二〇〇六年"是否改为"2006年"
- [ ] "打算"是否改为"决定"或"拟"
- [ ] "要求"是否改为"拟请"或"建议"
- [ ] "老百姓"是否改为"市民"或"群众"
- [ ] "一把手"是否改为"主要负责人"
- [ ] "没有着落"是否改为"缺口"

### 2. 平实性检查

- [ ] **朴实无华**：是否堆砌华丽辞藻
- [ ] **语意实在**：是否虚话套话较多
- [ ] **通俗易懂**：是否晦涩难懂
- [ ] **数据说话**：是否用数据代替形容词
- [ ] **避免溢美**：是否过度表扬
- [ ] **实事求是**：是否符合实际情况

#### 常见夸张用语核查

- [ ] 是否使用"英明领导"（应改为"正确领导"）
- [ ] 是否使用"举世瞩目"（应删除或改为"显著"）
- [ ] 是否使用"辉煌成就"（应改为"显著成绩"）
- [ ] 是否使用"排山倒海之势"（应删除）

### 3. 简明性检查

- [ ] **简洁明快**：是否开门见山
- [ ] **删繁就简**：是否可删除不必要的词语
- [ ] **短句为主**：是否使用过长句式
- [ ] **表意集中**：每段是否只有一层意思
- [ ] **避免重复**：有无重复表述
- [ ] **直截了当**：是否绕弯子

#### 需要删除的常用虚化词语

- [ ] "进行"（如：进行研究→研究）
- [ ] "对"（如：对问题进行分析→分析问题）
- [ ] "的"（适当删除，如：我们的同志→我单位同志）
- [ ] "了"（适当删除，如：完成了任务→完成任务）
- [ ] "工作"（适当删简，如：宣传工作→宣传）

### 4. 庄重性检查

- [ ] **书面语言**：是否使用规范书面语
- [ ] **避免口语**：是否使用方言、口语
- [ ] **避免网络语**：是否使用网络流行语
- [ ] **语气得体**：对上级、平级、下级语气是否得当
- [ ] **专用语规范**：公文专用语使用是否正确
- [ ] **用词稳妥**：是否使用绝对化、极端化词语

#### 需要避免的口语化表达

- [ ] "这事儿"（改为"此事"）
- [ ] "咱们"（改为"本单位"）
- [ ] "撸起袖子"（改为"大力"）
- [ ] "干货"（改为"实质性内容"）
- [ ] "拍脑袋"（改为"随意"）

### 5. 规范用语检查

#### 开头用语

- [ ] "根据"是否用于政策、法律依据
- [ ] "按照"是否用于执行、遵循
- [ ] "为"是否用于目的
- [ ] "关于"是否用于事由

#### 结尾用语

- [ ] 请示类："妥否，请批示"等使用是否正确
- [ ] 报告类："特此报告"等使用是否正确
- [ ] 通知类："特此通知"等使用是否正确
- [ ] 函类："特此函复""请予函复"等使用是否正确
- [ ] 禁止类："特此通告"等使用是否正确

#### 主要用语核查

- [ ] 依据类：根据、按照、遵照、依照
- [ ] 目的类：为、为了
- [ ] 承启类：现将、特作如下、综上所述
- [ ] 期请类：恳请、拟请、望予、请予
- [ ] 期复类：妥否、当否、可否
- [ ] 经办类：业经、已将、施行
- [ ] 谦敬类：承蒙、不胜

## 四、逻辑检查清单

### 1. 层次结构检查

- [ ] **层次分明**：是否层次清楚，条理分明
- [ ] **逻辑递进**：是否层层递进，环环相扣
- [ ] **因果关系**：因果关系是否成立
- [ ] **总分关系**：总分是否合理
- [ ] **主次关系**：主次是否分明，重点是否突出

### 2. 内容一致性检查

- [ ] **文理通顺**：是否读起来通顺流畅
- [ ] **前后呼应**：前后表述是否一致
- [ ] **首尾连贯**：开头与结尾是否连贯
- [ ] **避免矛盾**：前后有无矛盾之处
- [ ] **文题相符**：全文与标题是否一致

### 3. 论证充分性检查

- [ ] **论据充分**：是否有足够的论据
- [ ] **数据支撑**：是否有数据支持
- [ ] **事实依据**：是否有事实依据
- [ ] **政策依据**：是否有政策支持
- [ ] **推理合理**：推理是否合乎逻辑

---

## 五、细节检查清单

### 1. 文字准确性

- [ ] **错别字**：全文通读，检查错别字
- [ ] **漏字**：检查是否有漏字现象
- [ ] **多字**：检查是否有多余字
- [ ] **人名**：人名是否准确，特别是领导姓名
- [ ] **地名**：地名、单位名称是否准确
- [ ] **职务**：领导职务是否准确

### 2. 标点符号检查

- [ ] **句号**：是否使用全角句号"。"
- [ ] **逗号**：是否使用规范，有无一逗到底
- [ ] **冒号**：主送机关后是否为全角冒号
- [ ] **括号**：附注是否使用圆括号
- [ ] **引号**：引用文件是否使用书名号
- [ ] **顿号**：并列词语之间是否为顿号

### 3. 数字使用检查

- [ ] **成文日期**：是否使用阿拉伯数字全称
- [ ] **份数序号**：是否使用3号阿拉伯数字
- [ ] **序号**：是否不编虚位（如1不写01）
- [ ] **页码**：是否使用4号半角阿拉伯数字
- [ ] **标点**：阿拉伯数字与汉字标点是否混用

### 4. 署名与印章

- [ ] **发文机关署名**：是否与发文机关一致
- [ ] **日期**：是否标注完整
- [ ] **印章**：是否按规定加盖印章
- [ ] **骑年盖月**：印章位置是否符合要求
- [ ] **党政机关**：是否不加盖印章（根据规定）

### 5. 其他细节

- [ ] **联系人**：请示、函等是否注明联系人
- [ ] **联系电话**：联系人电话是否完整
- [ ] **密级标注**：是否标注密级（如需）
- [ ] **紧急程度**：是否标注紧急程度（如需）
- [ ] **主题词**：是否标注主题词（部分机关要求）
- [ ] **印发份数**：是否标注印发份数
- [ ] **发文机关代字**：是否正确

---

## 六、发文前最终确认

### 1. 审批流程

- [ ] **科室负责人审核**：是否已审核
- [ ] **分管领导审核**：是否已审核
- [ ] **主要领导签发**：是否已签发
- [ ] **会签部门意见**：是否已征求意见

### 2. 文件属性

- [ ] **发文机关**：是否正确
- [ ] **主送机关**：是否正确、完整
- [ ] **抄送机关**：是否需要抄送
- [ ] **密级**：是否需要标注
- [ ] **紧急程度**：是否需要标注
- [ ] **公开属性**：是否确定公开属性

### 3. 技术检查

- [ ] **页边距**：是否在打印预览中检查
- [ ] **分页**：是否检查有无孤行、寡行
- [ ] **打印效果**：是否打印预览检查整体效果
- [ ] **电子文件**：是否保存为PDF格式

### 4. 发送前确认

- [ ] **文件份数**：打印份数是否正确
- [ ] **附件材料**：附件是否齐全
- [ ] **发文登记**：是否登记发文簿
- [ ] **发文渠道**：是否选择正确发文渠道

## 七、快速检查口诀

**格式检查**：
天三七、左二八，版心一五六乘二二五；
正文三仿宋，标题二小标；
层次要分明，四层次序清；
页码四半角，单右双左分。

**内容检查**：
标题准，主送明；
文种对，格式清；
一文一事请示通；
逻辑顺，文理通。

**语言检查**：
准确四字要记牢：准平简庄。
用词准，語言平，
文字简，格调庄。

**整体检查**：
文经我手无差错，
事经我办请放心。

---

**使用前提醒**：
1. 本清单为基本检查项目，特殊公文可能有特殊要求
2. 重要公文建议多人交叉检查
3. 对于不确定的问题，应查阅相关标准或咨询专业人员
