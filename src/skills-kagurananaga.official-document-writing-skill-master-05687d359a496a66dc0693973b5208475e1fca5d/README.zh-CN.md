# 公文写作Skill项目

## 项目简介

这是一个专业的党政机关公文写作Skill工具包，严格遵循GB/T 9704-2012《党政机关公文格式》国家标准，提供完整的公文写作指导、模板、示例和质量检查清单。

本项目基于Claude Code官方的skill-creator框架进行优化，采用渐进式披露（progressive disclosure）设计理念，为公文写作者提供高效、专业的支持。

- **适用对象**：党政机关工作人员、企事业单位文秘人员、公文写作学习者

## 🎯 核心功能

### 1. 格式规范指导
按照GB/T 9704-2012国家标准，提供：
- 完整的公文格式要求（纸张、版面、字体、字号）
- 详细的排版规则（版头、主体、版记）
- 特定格式说明（信函、命令、纪要）
- 页码、表格、计量单位使用规范

### 2. 公文模板库
提供6种常用公文的标准模板和30+真实案例：
- ✅ 请示（3个示例）
- ✅ 通知（4个示例，含会议、培训、批转）
- ✅ 函（4个示例，含商洽、请批、答复）
- ✅ 总结（6个示例，含工作总结、经验总结）
- ✅ 会议纪要（2个示例，含党委会议、办公会议）
- ✅ 报告（1个详细示例）

### 3. 写作技巧指南
- 公文语言四大原则：准确、平实、简明、庄重
- 标题拟定"六看"标准
- 经验提炼"四步法"
- 80+常见错误修正对照
- 50+规范用语库

### 4. 质量检查清单
100+项质量检查点，涵盖：
- 格式检查（40+项）
- 内容检查（30+项，分5种文种）
- 语言检查（40+项）
- 逻辑检查（10+项）
- 细节检查（20+项）
- 发文前确认（10+项）

## 🚀 使用场景

### 场景1：新写一份公文
**适用**：需要从头撰写公文
**流程**：
1. 查阅SKILL.md确定文种
2. 导航至document-templates.md
3. 选择对应模板
4. 参考写作技巧填充内容
5. 使用quality-checklist.md检查

### 场景2：修改完善公文
**适用**：已有初稿，需要优化
**流程**：
1. 使用quality-checklist.md逐项检查
2. 发现问题标记
3. 查阅writing-techniques.md获取修改建议
4. 参考模板示例优化表达
5. 二次检查确认

### 场景3：检查格式规范
**适用**：需要检查公文格式是否正确
**流程**：
1. 打开quality-checklist.md
2. 按格式检查部分逐项核对
3. 对照gb-t-9704-2012-standard.md
4. 标注不符合项
5. 修正并复查

### 场景4：学习公文写作
**适用**：新手学习公文写作方法
**流程**：
1. 阅读SKILL.md了解整体框架
2. 学习writing-techniques.md四大原则
3. 研究document-templates.md案例
4. 掌握quality-checklist.md检查要点
5. 实践练习并总结经验

### 场景5：快速查询规范
**适用**：工作中快速查找具体规范
**方法**：
- 查格式→gb-t-9704-2012-standard.md
- 查模板→document-templates.md
- 查用语→writing-techniques.md
- 查检查项→quality-checklist.md
