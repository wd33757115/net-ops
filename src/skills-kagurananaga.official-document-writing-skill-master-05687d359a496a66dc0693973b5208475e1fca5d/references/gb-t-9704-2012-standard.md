## 标准概述

根据GB/T 9704-2012《党政机关公文格式》国家标准，本标准规定了党政机关公文通用的纸张要求、排版和印制装订要求、公文格式各要素的编排规则，并给出了公文的式样。

**适用范围**：各级党政机关制发的公文，其他机关和单位可参照执行。

**实施日期**：2012年7月1日

## 公文用纸要求

### 技术指标
- **纸张定量**：60-80 g/m² 的胶版印刷纸或复印纸
- **白度**：80%-90%</li>
- **不透明度**：≥85%
- **pH值**：7.5-9.5
- **横向耐折度**：≥15次

### 幅面尺寸
- **标准**：A4型纸（GB/T 148）
- **成品尺寸**：210mm×297mm

## 版面设置标准

### 页边与版心尺寸

| 参数 | 尺寸 | 说明 |
|------|------|------|
| 天头（上白边） | 37mm±1mm | 首页距离版心上边缘 |
| 订口（左白边） | 28mm±1mm | |
| 版心尺寸 | 156mm×225mm | |
| 行数 | 22行/面 | 撑满版心 |
| 字数 | 28字/行 | 撑满版心 |
| 行距 | 28磅 | |

### 文字颜色
- 无特殊说明，公文文字颜色均为**黑色**

### 制版要求
- 版面干净无底灰
- 字迹清楚无断划
- 尺寸标准，版心不斜
- 误差不超过1mm

### 印刷要求
- 双面印刷
- 页码套正
- 两面误差不超过2mm
- 油墨：黑色BL100%，红色Y80%|M80%

### 装订要求
- 左侧装订
- 不掉页
- 两页间误差不超过4mm
- 裁切后成品误差±2mm
- 四角成90°，无毛茬或缺损

**骑马订或平订要求**：
1. 订位为两钉外订眼距版面上下边缘各70mm处，允许误差±4mm
2. 无坏钉、漏钉、重钉，钉脚平伏牢固
3. 骑马订钉锯订在折缝线上，平订钉锯与书脊间距离3-5mm

## 字体字号规范

### 核心要素规范

| 要素 | 字体 | 字号 | 位置要求 |
|------|------|------|----------|
| **发文机关标志** | 小标宋体 | 自定 | 居中 |
| **文件标题** | 小标宋体 | 2号 | 居中 |
| **正文** | 仿宋_GB2312 | 3号 | 隔段左空二字 |
| **第一层次标题** | 黑体 | 3号 | 使用"一、" |
| **第二层次标题** | 楷体_GB2312 | 3号 | 使用"（一）" |
| **第三层次标题** | 仿宋_GB2312 | 3号 | 使用"1." |
| **第四层次标题** | 仿宋_GB2312 | 3号 | 使用"（1）" |
| **份数序号** | 黑体 | 3号 | 左上角第一行 |
| **密级和保密期限** | 黑体 | 3号 | 左上角第二行 |
| **紧急程度** | 黑体 | 3号 | 左上角第三行 |
| **发文字号** | 仿宋_GB2312 | 3号 | 红色分隔线下方居中 |
| **签发人"三字"** | 仿宋_GB2312 | 3号 | 右空一字 |
| **签发人姓名** | 楷体_GB2312 | 3号 | 签发人三字后 |
| **附件说明** | 黑体 | 3号 | 正文下空一行左空二字 |
| **附注** | 仿宋_GB2312 | 3号 | 成文日期下一行左空二字加圆括号 |
| **抄送机关** | 仿宋_GB2312 | 4号 | 印发机关上一行左右各空一字 |
| **印发机关和日期** | 仿宋_GB2312 | 4号 | 末条分隔线之上 |
| **页码** | 宋体（半角） | 4号 | 版心下边缘左右各一字线 |

### 结构层次序数

```
一、 第一层次（黑体）
  （一） 第二层次（楷体_GB2312）
    1.  第三层次（仿宋_GB2312）
      （1） 第四层次（仿宋_GB2312）
```

**层次标题位置要求**：
- 第一层次标题：单独成行
- 第二层次标题：单独成行
- 第三、四层次标题：可与正文连排

## 组成部分格式

### 一、版头部分（首页红色分隔线以上）

#### 1. 份号
- 使用6位3号**阿拉伯数字**
- 顶格编排在版心左上角第1行
- 例：000001

#### 2. 密级和保密期限
- 使用3号**黑体字**
- 顶格编排在版心左上角第2行
- 保密期限中的数字用阿拉伯数字标注
- 格式：密级★保密期限
- 例：机密★3年

#### 3. 紧急程度
- 使用3号**黑体字**
- 顶格编排在版心左上角
- 如需同时标注份号、密级和保密期限、紧急程度，按顺序自上而下分行排列

#### 4. 发文机关标志
- 使用**发文机关全称**或**规范化简称**加"文件"二字
- 也可使用发文机关全称或规范化简称
- 居中排布，上边缘至版心上边缘35mm
- 推荐使用**小标宋体字**，颜色为**红色**
- 联合行文时：
  - 如需同时标注联署发文机关名称，一般将**主办机关名称**排列在前
  - 如有"文件"二字，应当置于发文机关名称右侧

#### 5. 发文字号
- 由**机关代字、年份、序号**组成
- 编排在发文机关标志下空二行位置，居中排布
- **年份**用4位阿拉伯数字，用**六角括号"〔〕"**括入
- **序号**不加"第"字，不编虚位
- 格式示例：国办发〔2024〕15号
- **上行文**：发文字号居左空一字编排，与最后一个签发人姓名处在同一行

#### 6. 签发人
- 上行文需标注签发人
- 格式："签发人"三字+全角冒号+签发人姓名
- "签发人"三字使用3号**仿宋_GB2312**字体
- 签发人姓名使用3号**楷体_GB2312**字体
- 居右空一字，编排在发文机关标志下空二行位置
- 多个签发人：
  - 按照发文机关的排列顺序从左到右、自上而下依次均匀编排
  - 一般每行排两个姓名
  - 回行时与上一行第一个签发人姓名对齐

#### 7. 版头中的分隔线
- 发文字号之下4mm处居中印一条与**版心等宽的红色分隔线**
- 高度约为0.5mm-1mm

### 二、主体部分（首页分隔线以下至末页首条分隔线以上）

#### 1. 标题
- 使用2号**小标宋体字**（方正）
- 编排于红色分隔线下空二行位置
- 分一行或多行居中排布
- 回行要求：
  - 做到词意完整
  - 排列对称
  - 长短适宜
  - 间距恰当
  - **使用梯形或菱形排列**

**标题排列示例**：
```
    国务院批转《发展改革委关于2024年
    深化经济体制改革工作意见》的通知

    国务院批转《进出口管理委员会
    国家经济委员会关于建立中国工艺美术
    行业协会的报告》的通知
```

#### 2. 主送机关
- 使用3号**仿宋_GB2312**字体
- 编排于标题下空一行位置，居左顶格
- 回行时仍顶格
- 最后一个机关名称后标**全角冒号**
- 如主送机关名称过多导致公文首页不能显示正文，应当将主送机关名称**移至版记**

#### 3. 正文
- 公文首页**必须显示正文**
- 使用3号**仿宋_GB2312**字体
- 编排于主送机关名称下一行
- 每个自然段左空二字，回行顶格
- 文中结构层次序数：
  - "一、"（黑体）
  - "（一）"（楷体_GB2312）
  - "1."（仿宋_GB2312）
  - "（1）"（仿宋_GB2312）

**段落格式**：
- 开头：引言部分，交代背景、目的、依据
- 主体：核心内容，分条列项阐述
- 结尾：总结、要求、希望等

#### 4. 附件说明
- 使用3号**仿宋_GB2312**字体
- 正文下空一行左空二字编排"附件"二字
- 后标全角冒号和附件名称
- **多个附件**：
  - 使用阿拉伯数字标注附件顺序号
  - 如"附件：1. XXXXX"
  - 附件名称后不加标点符号
- **回行要求**：与上一行附件名称的首字对齐

#### 5. 发文机关署名、成文日期和印章

##### 5.1 加盖印章的公文
- **单一机关行文时**：
  - 一般在成文日期之上、以成文日期为准**居中编排发文机关署名**
  - 印章端正、居中下压发文机关署名和成文日期
  - 使发文机关署名和成文日期居印章中心偏下位置
  - 印章顶端应当上距正文（或附件说明）一行之内（3号字）

- **联合行文时**：
  - 各发文机关署名按发文机关顺序整齐排列在相应位置
  - 将印章一一对应、端正、居中下压发文机关署名
  - 最后一个印章端正、居中下压发文机关署名和成文日期
  - 印章之间排列整齐、互不相交或相切
  - 每排印章两端不得超出版心
  - 首排印章顶端上距正文（或附件说明）一行之内

##### 5.2 不加盖印章的公文
- **单一机关行文时**：
  - 在正文（或附件说明）下空一行右空二字编排发文机关署名
  - 在发文机关署名下一行编排成文日期
  - 成文日期首字比发文机关署名首字右移二字
  - 如成文日期长于发文机关署名，成文日期右空二字编排

- **联合行文时**：
  - 先编排主办机关署名，其余发文机关署名依次向下编排

##### 5.3 加盖签发人签名章的公文
- 使用红色签名章
- **单一机关制发时**：
  - 在正文（或附件说明）下空二行右空四字加盖签名章
  - 签名章左空二字标注签发人职务
  - 以签名章为准上下居中排布
  - 在签发人签名章下空一行右空四字编排成文日期

##### 5.4 成文日期中的数字
- 使用**阿拉伯数字**
- 年份应标**全称**（如：2024年）
- 月、日不编虚位（如：5月8日，不写05月08日）
- 格式：2024年5月8日

##### 5.5 特殊情况说明
- 当公文排版后所剩空白处不能容下印章或签发人签名章、成文日期时，可以采取**调整行距、字距**的措施解决
- **不得出现空白印章**

#### 6. 附注
- 使用3号**仿宋_GB2312**字体
- 居左空二字加**圆括号**编排在成文日期下一行
- 内容：公文印发传达范围等需要说明的事项
- 例：（联系人：张三，联系电话：12345678901）

#### 7. 附件
- 附件应当**另面编排**，并在版记之前，与公文正文一起装订
- "附件"二字及附件顺序号使用3号**黑体字**顶格编排在版心左上角第一行
- 附件标题居中编排在版心第三行
- 附件顺序号和附件标题应当与附件说明的表述一致
- 附件格式要求同正文
- 版记位于附件之后

**附件与正文分开装订时**：
- 在附件左上角第一行顶格编排公文的发文字号
- 在发文字号后标注"附件"二字及附件顺序号

### 三、版记部分（末页首条分隔线以下、末条分隔线以上）

#### 1. 版记中的分隔线
- 分隔线与**版心等宽**
- **首条分隔线**和**末条分隔线**用粗线（推荐高度0.35mm）
- **中间分隔线**用细线（推荐高度0.25mm）
- 首条分隔线位于版记中第一个要素之上
- 末条分隔线与公文最后一面的版心下边缘重合

#### 2. 抄送机关
- 使用4号**仿宋_GB2312**字体
- 在印发机关和印发日期之上一行、左右各空一字编排
- "抄送"二字后加**全角冒号**和抄送机关名称
- 回行时与冒号后的首字对齐
- 最后一个抄送机关名称后标**句号**

**主送机关移至版记时**：
- 将"抄送"改为"主送"
- 编排方法同抄送机关

**同时有主送和抄送时**：
- 主送机关置于抄送机关之上一行
- 之间不加分隔线

#### 3. 印发机关和印发日期
- 使用4号**仿宋_GB2312**字体
- 编排在末条分隔线之上
- 印发机关左空一字，印发日期右空一字
- 印发日期用阿拉伯数字将年、月、日标全
- 后加"印发"二字

**版记中如有其他要素**（如印发份数）：
- 应当将其与印发机关和印发日期用一条细分隔线隔开

## 页码编排规则

### 1. 基本格式
- 使用4号半角**宋体阿拉伯数字**
- 编排在公文版心下边缘之下
- **单页码**：居右空一字
- **双页码**：居左空一字
- 数字左右各放一条**一字线**（占一个字符位置）
- 一字线上距版心下边缘7mm

### 2. 特殊页码编排
- **版记页前有空白页**：
  - 空白页和版记页均不编排页码
- **公文附件与正文一起装订**：
  - 页码应当连续编排
- **横排表格**（A4纸型表格横排）：
  - 页码位置与公文其他页码保持一致
  - 单页码，表头在订口一边
  - 双页码，表头在切口一边

### 3. 页码字体和字号
- 阿拉伯数字使用**半角宋体**
- 字号为**4号**

## 公文中表格编排

### 1. 横排表格
- **页码位置**与公文其他页码保持一致
- 单页码时：表头在**订口**（左侧）一边
- 双页码时：表头在**切口**（右侧）一边

### 2. 表格字体要求
- 表格中文字使用**仿宋_GB2312**字体
- 字号可根据表格内容适当调整，一般为**3号或小3号**

## 公文中计量单位、标点符号和数字用法

### 1. 计量单位
- 应符合GB 3100、GB 3101和GB 3102（所有部分）
- 使用国际单位制（SI）单位

### 2. 标点符号
- 应符合GB/T 15834《标点符号用法》
- 主要要求：
  - 使用**全角标点符号**
  - 句末用**全角句号"。"**
  - 列举用**全角顿号"、"**
  - 数字间的间隔用**全角空格**或**半角空格**（根据需要）

### 3. 数字用法
- 应符合GB/T 15835《出版物上数字用法》

**主要规则**：
- **使用阿拉伯数字的情况**：
  - 公历世纪、年代、年、月、日和时刻
  - 计数与计量（包括正负整数、分数、小数、百分比、约数等）
  - 代号、代码和序号
  - 物理量量值

- **使用汉字数字的情况**：
  - 数字作为词素构成定型的词、词组、成语、惯用语、缩略语或具有修辞色彩的词语
  - 邻近的两个数字并列连用表示概数（如：三五天）
  - 星期几
  - 夏历和中国清代以前历史纪年

**公文成文日期**：一律使用**阿拉伯数字**标注

## 公文的特定格式

### 1. 信函格式

**发文机关标志**：
- 使用发文机关全称或规范化简称
- 居中排布，上边缘至上页边为30mm
- 推荐使用红色小标宋体字
- 联合行文时，使用主办机关标志

**线框**：
- 发文机关标志下4mm处：红色双线（上粗下细）
- 距下页边20mm处：红色双线（上细下粗）
- 线长均为170mm，居中排布

**序号和密级**：
- 如需标注份号、密级和保密期限、紧急程度
- 顶格居版心左边缘编排在第一条红色双线下
- 按顺序自上而下分行排列
- 第一个要素与该线的距离为3号汉字高度的7/8

**发文字号**：
- 顶格居版心右边缘编排在第一条红色双线下
- 与该线的距离为3号汉字高度的7/8

**标题**：
- 居中编排，与其上最后一个要素相距二行

**首页**：
- **不显示页码**

**版记**：
- 版记不加印发机关和印发日期、分隔线
- 位于公文最后一面版心内最下方

### 2. 命令（令）格式

**发文机关标志**：
- 由发文机关全称加"命令"或"令"字组成
- 居中排布，上边缘至版心上边缘为20mm
- 推荐使用红色小标宋体字

**令号**：
- 居中编排
- 位于发文机关标志下空二行

**正文**：
- 位于令号下空二行编排

**签发人职务、签名章和成文日期**：
- 编排见7.3.5.3规定（引用标准原文）

### 3. 纪要格式

**纪要标志**：
- 由"XXXXX纪要"组成
- 居中排布，上边缘至版心上边缘为35mm
- 推荐使用红色小标宋体字

**出席人员名单**：
- 使用3号黑体字
- 在正文或附件说明下空一行左空二字编排"出席"二字
- 后标全角冒号
- 冒号后用3号仿宋_GB2312字体标注出席人单位、姓名
- 回行时与冒号后的首字对齐

**请假和列席人员名单**：
- 依次另起一行
- 将"出席"二字改为"请假"或"列席"
- 编排方法同出席人员名单

**自定义**：
- 纪要格式可以根据实际制定

## 公文排版检查要点

### 首页检查项目
- [ ] 份号、密级、紧急程度顺序是否正确
- [ ] 发文机关标志位置是否正确（上边缘距版心上边缘35mm）
- [ ] 发文字号位置是否正确
- [ ] 有签发人时格式是否规范
- [ ] 红色分隔线位置是否正确（发文字号下4mm）
