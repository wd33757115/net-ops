# 07 系统配置说明

> 版本：2026-05-24  
> 配置加载：仓库根目录 `.env`（FastAPI / Celery / Django 共用）

---

## 1. 配置来源

| 组件 | 加载方式 | 文件 |
|------|----------|------|
| FastAPI / Celery | Pydantic Settings | `src/common/config.py` |
| Django BFF | `os.getenv` + dotenv | `web/django_backend/django_backend/settings.py` |
| React | Vite env | `web/react_frontend/.env` |
| Docker Compose | `environment` / `env_file` | `deployment/docker-compose.yml` |

### 2.4 Celery

| 变量 | 默认 | 说明 |
|------|------|------|
| `CELERY_BROKER_URL` | `redis://localhost:6379/0` | 消息 broker（生产常用 RabbitMQ） |

```env

CELERY_RESULT_BACKEND=redis://localhost:6379/1
```

## 5. Docker Compose 覆盖

`deployment/docker-compose.yml` 中为各服务注入环境变量，典型覆盖：

- FastAPI：`ENFORCE_BFF_ORIGIN=true`
- PostgreSQL：容器内 hostname `postgres`
- Langfuse：独立 PostgreSQL + 初始化

修改 Compose 后需 `docker compose up -d` 重建。

---

## 6. 配置优先级

1. 进程环境变量（最高）
2. 仓库根 `.env`
3. `web/django_backend/.env`（Django 本地覆盖）
4. Pydantic / Django 代码默认值
